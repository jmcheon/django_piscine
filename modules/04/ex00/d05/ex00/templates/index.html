Markdown Cheatsheet

Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

Emphasis

*Italics* or _Italics_
**Bold** or __Bold__
***Bold Italics*** or ___Bold Italics___

Lists

Unordered

* Item 1
* Item 2
- Item 3
+ Item 4

Ordered

1. First item
2. Second item
3. Third item

Links

[Link text](URL)

Images

![Alt text](image.jpg)

Code

Inline code: code
Code block:

print("Hello, world!")

Horizontal Rule

---
***
___

Blockquotes

> This is a blockquote.

Escaping Characters

Use a backslash \ to escape characters like *, _, #, etc.

Tables

| Header 1 | Header 2 |
| -------- | -------- |
| Row 1, Col 1 | Row 1, Col 2 |
| Row 2, Col 1 | Row 2, Col 2 |
